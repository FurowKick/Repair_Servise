# Система управления заявками на ремонт

Desktop-приложение для автоматизации учёта заявок в сервисном центре бытовой техники.

## 📦 Стек технологий
- **C# / .NET 8** (WinForms)
- **PostgreSQL 14+**
- **Npgsql** (ADO.NET драйвер)
- **DBeaver / pgAdmin** (для администрирования БД)

## 🚀 Быстрый старт

### 1. Требования
- Windows 10/11
- Установленный **PostgreSQL** (сервер должен быть запущен)
- .NET 8 Runtime (приложение скомпилировано как Self-Contained, но на всякий случай)
